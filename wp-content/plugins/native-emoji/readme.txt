Insert emojis in your posts, pages, and custom post types

== Description ==

This is not just a plugin, this is the plugin for use emoji in a native way. When activated you will see a new button in your wordpress editor, from there you will be able to include more than 1,000 icons in to your posts, pages, and custom posts types

If the Operative System doesn't support emoji, this plugin insert an image instead of the emoji code. 

= Features =

* Very Easy To Use
* More Than 1,000 emojis
* Supports Any Theme
* Supports Custom Posts Types
* iOS Native Emoji
* Android Native Emoji

Read more about this plugin [here](http://native-emoji.davabuu.com/)

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Native Emoji'
3. Activate Native Emoji from your Plugins page.
4. Visit Any post type from your admin and click on the smiley button in the editor

= From WordPress.org =

1. Download Native Emoji.
2. Upload the 'Native Emoji' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Native Emoji from your Plugins page. 
4. Visit Any post type from your admin and click on the smiley button in the editor
